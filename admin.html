File Format Instructions:

Create a CSV file with these columns:

  • question - The question text
  • option1, option2, option3, option4 - The four answer options
  • correct_answer - Number (1-4) of the correct option
  • category - Quiz category (e.g., "Technical", "General")
Example: 
question,option1,option2,option3,option4,correct_answer,category
"What does HTML stand for?","Hyper Text Markup Language","High Tech Modern Language","Hyper Transfer Markup Language","Home Tool Markup Language",1,"Technical"
"Which language runs in browsers?","Python","Java","JavaScript","C++",3,"Technical"
